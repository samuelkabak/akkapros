# Support for akkapros

Thank you for using akkapros! This document explains how to get help when you need it.

## 📋 Before Asking for Help

Please check these resources first—your question may already be answered:

- **[README.md](README.md)** – Overview and quick start
- **[GETTING_STARTED.md](docs/GETTING_STARTED.md)** – Step-by-step tutorial
- **[metrics-computation.md](docs/akkapros/metrics-computation.md)** – Detailed explanation of all metrics
- **[Existing Issues](https://github.com/samuelkabak/akkapros/issues)** – Search to see if someone already had your question

## 🐛 Reporting Bugs

If you've found a bug:

1. **Check existing issues** to make sure it hasn't already been reported
2. **Create a new issue** using the "Bug report" template
3. **Include**:
   - Your operating system (Windows/macOS/Linux)
   - Python version
   - Exact command you ran
   - Full error message (if any)
   - Sample input that reproduces the problem

## 💡 Feature Requests

Have an idea for improvement?

1. **Check existing issues** for similar suggestions
2. **Create a new issue** using the "Feature request" template
3. **Describe**:
   - What you'd like to accomplish
   - Why it would be useful
   - Any examples from other tools

## ❓ Questions About Usage

For questions about how to use akkapros:

1. **Check the documentation** first (linked above)
2. **Search existing issues** for similar questions
3. **Open a new issue** with the question tag
