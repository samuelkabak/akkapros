# 🎉 Announcing akkapros v1.0.0 – Akkadian Prosody Toolkit

**March 14, 2026**

I am pleased to announce the first stable release of **akkapros**, an open-source Python toolkit for computational prosody reconstruction in Akkadian. This project addresses a fundamental question in Assyriology: how were the lexical stress rules of Akkadian realized in connected speech?

After extensive development, testing on a corpus of 4,917 words from Standard Babylonian literary texts, and refinement based on acoustic metrics, the toolkit is now ready for use by the research community.

---

## 📦 What's Included

| Tool | Purpose |
|------|---------|
| `atfparser.py` | Converts eBL ATF files to clean phonological text |
| `syllabify.py` | Syllabifies Akkadian text following Huehnergard (2011) |
| `prosmaker.py` | Applies the prosody realization algorithm |
| `metricalc.py` | Computes acoustic metrics (%V, ΔC, VarcoC) |
| `fullprosmaker.py` | End-to-end pipeline (syllabify → prosody → metrics → print) |
| `printer.py` | Converts output to multiple formats (acute, bold, IPA, XAR) |

---

## 🔬 Key Features

- **Two accent styles**: LOB (Literary Old Babylonian) and SOB (Standard Old Babylonian)
- **Phonologically grounded operations**: vowel lengthening and consonant gemination
- **Intelligent merging**: handles function words and prosodic units
- **Diphthong pipeline**: split during syllabification, restored after processing
- **Comprehensive metrics**: %V, ΔC, VarcoC with pause ratio sensitivity (30%, 35%, 40%)
- **Multiple output formats**:
  - Acute-accented text for scholarly notation
  - Bolded syllables for visual inspection
  - IPA with prosodic markers for phonetic analysis
  - XAR orthography for specialized applications
- **MBROLA integration**: 878-word diphone recording script included

---

## 📊 Validation

The algorithm has been validated on a corpus of 4,917 words from three Standard Babylonian literary texts:
- Enūma Eliš (tablets II, IV, VI, VII)
- Erra and Išum (tablet I)
- Marduk's Address to the Demons

**Key results:**
- Original VarcoC: 69.09 (compatible with stress-timing)
- Prominence rate: 13.63% (emerges from data, not design)
- Repaired VarcoC: 70.67 (remains consistent with stress-timing)
- 49.9% of words participate in prosodic merging

All results are fully reproducible using the included demo scripts.
